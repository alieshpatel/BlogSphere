# MERN Blog App

A simple full-stack Blog Application built using the MERN Stack (MongoDB, Express, React, Node.js).  
This project is designed for beginners to understand CRUD operations and full-stack development.

## Features

- Create a new blog post
- View all blog posts
- View single blog details
- Edit blog post
- Delete blog post
- REST API integration
- MongoDB database connection

---

## Tech Stack

### Frontend
- React (Vite)
- Axios
- CSS / Tailwind (optional)

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
